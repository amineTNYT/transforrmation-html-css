CSS Transformations

CSS transformations allow you to modify the shape, size, position, or orientation of an HTML element without affecting the document flow. Explore the different transformation functions with these interactive examples.

Try the Interactive Playground →

1. The translate() Function

Moves an element along the X (horizontal) and Y (vertical) axes.

/* Syntax */
transform: translateX(50px); /* Horizontal movement */
transform: translateY(30px); /* Vertical movement */
transform: translate(50px, 30px); /* Combined movement */
translate()

2. The rotate() Function

Rotates an element by a specified angle in degrees.

/* Syntax */
transform: rotate(45deg);   /* 45 degree rotation */
transform: rotate(-90deg);  /* -90 degree rotation */
transform: rotate(180deg);  /* 180 degree rotation */
rotate()

3. The scale() Function

Changes the size of an element (enlarges or shrinks).

/* Syntax */
transform: scale(1.5);        /* Enlarges by 50% */
transform: scale(0.5);        /* Shrinks by 50% */
transform: scale(1.5, 0.8);   /* Different X and Y values */
scale()

4. The skew() Function

Skews an element along the X and Y axes.

/* Syntax */
transform: skewX(20deg);     /* Horizontal skew */
transform: skewY(10deg);     /* Vertical skew */
transform: skew(20deg, 10deg); /* Combined skew */
skew()

5. Combined Transformations

You can combine multiple transformations on the same element.

/* Syntax */
transform: translate(50px, 30px) 
           rotate(45deg) 
           scale(1.2);

/* The order of transformations is important! */
