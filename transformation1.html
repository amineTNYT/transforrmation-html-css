Transformations CSS

Les transformations CSS vous permettent de modifier la forme, la taille, la position ou l'orientation d'un élément HTML sans affecter le flux du document. Explorez les différentes fonctions de transformation avec ces exemples interactifs.

1. La Fonction translate()

Déplace un élément le long des axes X (horizontal) et Y (vertical).

/* Syntaxe */
transform: translateX(50px); /* Déplacement horizontal */
transform: translateY(30px); /* Déplacement vertical */
transform: translate(50px, 30px); /* Déplacement combiné */
translate()

2. La Fonction rotate()

Fait tourner un élément d'un angle spécifié en degrés.

/* Syntaxe */
transform: rotate(45deg);   /* Rotation de 45 degrés */
transform: rotate(-90deg);  /* Rotation de -90 degrés */
transform: rotate(180deg);  /* Rotation de 180 degrés */
rotate()

3. La Fonction scale()

Modifie la taille d'un élément (agrandit ou réduit).

/* Syntaxe */
transform: scale(1.5);        /* Agrandit de 50% */
transform: scale(0.5);        /* Réduit de 50% */
transform: scale(1.5, 0.8);   /* Valeurs X et Y différentes */
scale()

4. La Fonction skew()

Incline un élément le long des axes X et Y.

/* Syntaxe */
transform: skewX(20deg);     /* Inclinaison horizontale */
transform: skewY(10deg);     /* Inclinaison verticale */
transform: skew(20deg, 10deg); /* Inclinaison combinée */
skew()

5. Transformations Combinées

Vous pouvez combiner plusieurs transformations sur le même élément.

/* Syntaxe */
transform: translate(50px, 30px) 
           rotate(45deg) 
           scale(1.2);

/* L'ordre des transformations est important ! */
