Les Transformations CSS

Les transformations CSS permettent de modifier la forme, la taille, la position ou l'orientation d'un élément HTML sans affecter le flux du document. Explorez les différentes fonctions de transformation avec ces exemples interactifs.

1. La fonction translate()

Déplace un élément sur les axes X (horizontal) et Y (vertical).

/* Syntaxe */
transform: translateX(50px); /* Déplacement horizontal */
transform: translateY(30px); /* Déplacement vertical */
transform: translate(50px, 30px); /* Déplacement combiné */
translate()

2. La fonction rotate()

Fait pivoter un élément selon un angle spécifié en degrés.

/* Syntaxe */
transform: rotate(45deg);   /* Rotation de 45 degrés */
transform: rotate(-90deg);  /* Rotation de -90 degrés */
transform: rotate(180deg);  /* Rotation de 180 degrés */
rotate()

3. La fonction scale()

Modifie la taille d'un élément (agrandit ou rétrécit).

/* Syntaxe */
transform: scale(1.5);        /* Agrandit de 50% */
transform: scale(0.5);        /* Rétrécit de 50% */
transform: scale(1.5, 0.8);   /* Différentes valeurs X et Y */
scale()

4. La fonction skew()

Incline un élément selon les axes X et Y.

/* Syntaxe */
transform: skewX(20deg);     /* Inclinaison horizontale */
transform: skewY(10deg);     /* Inclinaison verticale */
transform: skew(20deg, 10deg); /* Inclinaison combinée */
skew()

5. Combinaison de Transformations

On peut combiner plusieurs transformations sur un même élément.

/* Syntaxe */
transform: translate(50px, 30px) 
           rotate(45deg) 
           scale(1.2);

/* L'ordre des transformations est important ! */
Combinaison
